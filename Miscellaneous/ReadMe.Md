YOUTHCONECT SA Website

This website was built using HTML, CSS and JavaScript.

The website includes: 
- A /*homepage*/ with the highight of career counseling, skills training, mentorship and latest news.
- An /*about us*/ page displaying our vision, our team and our partners.
- A /*programmes*/ page showing our programmes, enterpreneurship and registration section.
- An /*events*/ page with the upcoming events.
- A /*blog*/ for our blog.
- A /*donate*/ page for supporting our mission.
- A /*contactus*/ page for contact information and details.
